##### GuttingPaint ##############################################
Version: 1.0

This html/javascript application is a rough clone of the classic
MS Paint, but for modern browsers.  At present it contains the
following features:

General Features:
- color picker: currently using browser's built-in color picker
- clear canvas: remove all painting actions from canvas
- undo/redo: ability to step backward/forward through all painting
  actions, redo queue is cleared on new painting action

Painting Actions:
- brush stroke: draw by holding right mouse down over canvas and moving mouse
  - brush size slider: select brush size from 2-50 in increments of 2
